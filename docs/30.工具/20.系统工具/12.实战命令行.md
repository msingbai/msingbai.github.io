## `curl` 的使用

### `curl` 基础请求

``` bash
curl https://xingcxb.com
```

### `curl` 代理请求

``` bash
# 方式一
# 不带密码
curl -x "http://127.0.0.1:1234" "https://xingcxb.com"
# 带密码
curl -x "http://user:pwd@127.0.0.1:1234" "https://xingcxb.com"
```
