# CSS基础

## 什么是 CSS？

- *CSS* 指的是**层叠样式表*** (*C*ascading *S*tyle *S*heets)
- HTML负责定义页面的结构和内容，CSS负责控制页面的外观和样式
- 通过CSS，你可以指定页面中各个元素的颜色、字体、大小、间距、边框、背景等样式，从而实现更精确的页面设计。

## CSS语法

通常由选择器、属性和属性值组成，多个规则可以组合在一起，以便同时应用多个样式

```css
选择器｛
属性1：属性值1;
属性2：属性值2;
}
```

例子

```css
/*这是一个p标签选择器*/
p{
	color: blue;
	font-size: 16px;
}
```

## CSS的三种导入方式

1. 内联样式  
2. 内部样式表 <style标签>
3. 外部样式表
